# Trabalho-Calculadora

## Como utiliza a Calculadora?

Para utilizá-la, acesse a pasta DIST e dê dois cliques sobre o arquivo Calculadora_Launcher.jar. com isso, será executado e a calculadora será apresentada na tela. 

## C (LIMPA TUDO)

## - (SUBTRAÇÃO)

## + (ADIÇÃO)

## / (DIVISÃO)

## * (MULTIPLICAÇÃO)

## = (IGUAL)

Caso tente dividir qualquer número por 0 vai aparecer na tela Inválido!!!, sendo necessário utilizar o botão C para reiniciar o cálculo.
